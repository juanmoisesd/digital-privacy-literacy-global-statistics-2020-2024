## Description

This dataset compiles global statistics on digital privacy literacy and data protection behaviors during 2020–2024. It covers awareness levels, privacy-protective behaviors, regulatory compliance awareness, and trends across regions and demographics.

Sources include international surveys, regulatory reports, and data from privacy advocacy organizations.
